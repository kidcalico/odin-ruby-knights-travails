# The Odin Project: Knights Travails

Given sufficient moves, a knight can move to every square on a chess board. This is a ruby implementation of an algorithm which calculates the shortest path from a given square to any other square on the board. 
---
For example:
`knight_moves([0,0],[3,3])` will return `[[0,0],[1,2],[3,3]]` or `[[0,0],[2,1],[3,3]]`
