# The Odin Project: Knights Travails
---

This is a ruby implementation of a breadth first algorithm, which calculates the shortest path from a given square to any other square on the board.

For example:
`knight_moves([0,0],[3,3])` will return
```
Initial location: [0, 0]
Target location: [3, 3]
Congratulations! You made it in 2 moves! Here's your path:
[0, 0]
[1, 2]
[3, 3]
```
